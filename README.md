# 🛡️ CyberX

A modern multi-page website exploring the connection between Cybersecurity and Artificial Intelligence.

This project was developed as a collaborative team project using HTML, CSS, JavaScript, and Bootstrap 5. It focuses on how Artificial Intelligence is transforming cybersecurity, improving threat detection, and helping protect digital systems from modern cyber threats.

## 📌 About The Project

CyberX is an educational and informative multi-page website designed to introduce users to the relationship between Artificial Intelligence and Cybersecurity.

The website provides information about modern cyber threats, AI-powered security systems, cybersecurity concepts, and the future of AI in digital security.

Our goal was to create a modern, responsive, and user-friendly website while working collaboratively as a team.
